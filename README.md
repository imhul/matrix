## Matrix Rain Effect

This project creates a Matrix rain effect using PixiJS and TypeScript, created for fun.

### Demo

You can view a [live demo here](https://imhul.github.io/matrix/)

### Features

- Realistic Matrix rain animation
- Customizable colors and effects
- Glow filter for enhanced visual appeal

### Installation

1. git clone https://github.com/imhul/matrix.git
2. cd matrix
3. npm install or yarn
4. npm run dev or yarn dev

### Usage

- keys 'r', 'R': Reset the matrix rain effect
- keys 'Esc', 's', 'S': Open/Close settings panel
